# Holochain Gym

Go to the [gym](https://holochain-gym.github.io).

## Contributor Setup

First, enter the nix-shell:

```
nix-shell
```

Then, install the dependencies:

```
npm install
```

## Starting

```
npm start
```

This will start a server with hot reload that should open automatically.

## Deploying to github pages

```
npm run gh-pages
```
